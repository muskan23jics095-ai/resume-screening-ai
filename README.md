Resume Screening AI

AI-powered Resume Screening System that predicts job role, calculates resume score, and identifies missing skills.

Features
Resume Upload (PDF)
Job Description Matching
Resume Score (0–100)
Missing Skills Detection
Role Prediction (Data Scientist / Analyst / Web Dev)
Interactive Dashboard
Tech Stack
Python
NLP
Scikit-learn
Streamlit
Plotly
How to Run

pip install -r requirements.txt
streamlit run app.py

Example Output

Predicted Role: Data Scientist
Resume Score: 82%
Missing Skills: SQL, Statistics
Matching Skills: Python, ML, Pandas
